## Описание ##
Учебный проект.
Сборник REST-сервисов job4j
#### Сервис auth ####
REST-сервис авторизации пользователей
#### Сервис employees ####
REST-сервис сотрудников.
Он связан через RestTemplate с сервисом авторизации пользователей.
#### Сервис chat ####
REST-сервис чат пользователей.
#### Технологии проекта ####
PostgreSQL
Maven
SpringBot
